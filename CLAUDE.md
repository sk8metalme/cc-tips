# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## プロジェクト概要

cc-tips は Claude Code の活用Tips、設定例、ベストプラクティスを集約するナレッジベースリポジトリです。

## リポジトリ構造（予定）

```
cc-tips/
├── tips/               # カテゴリ別Tipsドキュメント
├── examples/           # 設定ファイルやスクリプトの実例
├── templates/          # 再利用可能なテンプレート
└── docs/               # 詳細ドキュメント
```

## 開発設定

このリポジトリは `outputStyle: "Learning"` で設定されており、Claude Codeが教育的なインサイトを提供しながら作業を進めます。

## コンテンツ作成ガイドライン

- Tipsは具体的で再現可能な形式で記述
- 設定例には使用場面とトレードオフを明記
- Markdownの一貫したフォーマットを維持
